Owly project!
===================

**Install raspberry system**

download link : http://downloads.raspberrypi.org/raspbian_latest

install : 

`sudo dd bs=1M if=2014-12-24-wheezy-raspbian.img of=/dev/mmcblk0`

**user** : pi

**password** : raspbian

**On raspberry**

`git clone https://github.com/k-yak/owly`

`chmod +x install.sh`

`./install.sh`
